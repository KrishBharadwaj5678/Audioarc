<h1 align="center"><img src="https://cdn.jsdelivr.net/gh/twitter/twemoji@latest/assets/svg/1f3b5.svg" width="35"/> Audiioarc</h1>

<p align="center">
  Audioarc это адаптивный музыкальный проигрыватель, созданный с использованием HTML, CSS и JavaScript. Он обеспечивает плавное прослушивание благодаря элегантному интерфейсу, удобным элементам управления и дизайну, оптимизированному для любого размера экрана.
</p>

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/features.gif" width="35"/> Возможности

|  **Возможность**                                     |  **Описание**                                                   |
| ------------------------------------------------ | ------------------------------------------------------------------ |
| 🎧 **Современный интерфейс музыкального плеера**             | Стильный и привлекательный дизайн для комфортного прослушивания музыки.              |
| 📱 **Полностью адаптивный дизайн**             | Идеально работает на всех устройствах и размерах экранов.               |
| ⏯️ **Расширенные элементы управления воспроизведением**          | Полный контроль над плейлистом с помощью интуитивно понятных кнопок.        |
| 🕒 **Динамическое отслеживание прогресса**          | Отображает текущее время воспроизведения и прогресс трека в реальном времени.        |
