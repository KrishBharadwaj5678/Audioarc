<h1 align="center"><img src="https://cdn.jsdelivr.net/gh/twitter/twemoji@latest/assets/svg/1f3b5.svg" width="35"/> Audiioarc</h1>

<p align="center">
  Audioarcは、HTML、CSS、JavaScriptで作成されたレスポンシブな音楽プレーヤーです。洗練されたインターフェース、スムーズな操作性、あらゆる画面サイズに最適化されたデザインにより、快適な音楽体験を提供します。
</p>

<img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/divider.gif" width="100%"/>

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/features.gif" width="35"/> 主な機能

|  **機能**                                     |  **説明**                                                   |
| ------------------------------------------------ | ------------------------------------------------------------------ |
| 🎧 **モダンな音楽プレーヤーインターフェース**             | 快適な音楽体験を実現するスタイリッシュで魅力的なレイアウト。              |
| 📱 **完全レスポンシブデザイン**             | すべてのデバイスや画面サイズで快適に動作します。               |
| ⏯️ **高度な再生コントロール**          | 直感的なボタンでプレイリストを完全に操作できます。        |
| 🕒 **動的な再生進行状況表示**          | 再生時間やトラックの進行状況をリアルタイムで表示します。        |
