<h1 align="center"><img src="https://cdn.jsdelivr.net/gh/twitter/twemoji@latest/assets/svg/1f3b5.svg" width="35"/> Audiioarc</h1>

<p align="center">
  O Audioarc é um reprodutor de música responsivo desenvolvido com HTML, CSS e JavaScript. Ele oferece uma experiência de audição fluida com uma interface elegante, controles intuitivos e um design otimizado para todos os tamanhos de tela.
</p>

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/features.gif" width="35"/> Funcionalidades

|  **Funcionalidade**                                     |  **Descrição**                                                   |
| ------------------------------------------------ | ------------------------------------------------------------------ |
| 🎧 **Interface Moderna de Reprodutor de Música**             | Layout elegante e atraente para uma experiência musical perfeita.              |
| 📱 **Design Totalmente Responsivo**             | Funciona perfeitamente em todos os dispositivos e tamanhos de tela.               |
| ⏯️ **Controles Avançados de Reprodução**          | Controle completo da sua playlist com botões intuitivos.        |
| 🕒 **Acompanhamento Dinâmico de Progresso**          | Exibe o tempo de reprodução e o progresso da faixa em tempo real.        |

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/techStack.gif" width="35"/> Stack Tecnológica

| Tecnologia | Objetivo |
|------|---------|
| <img src="https://skillicons.dev/icons?i=html" width="25"/> **HTML5** | Estrutura das páginas web e integração do elemento de áudio. |
| <img src="https://skillicons.dev/icons?i=css" width="25"/> **CSS3** | Estilização, animações e criação de layouts responsivos. |
| <img src="https://skillicons.dev/icons?i=js" width="25"/> **JavaScript** | Gerencia os controles de música, atualizações de progresso e interatividade. |
| <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/json/json-original.svg" width="25"/> **JSON** | Armazena e gerencia dados das músicas, como título, artista e caminhos dos arquivos. |
