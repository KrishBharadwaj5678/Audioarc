<p align="center">
  Audioarc is a responsive music player crafted with HTML, CSS, and JavaScript. It delivers a smooth listening experience with an elegant interface, seamless controls, and a design optimized for every screen size.
</p>

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/features.gif" width="35"/> Features

|  **Feature**                                     |  **Description**                                                   |
| ------------------------------------------------ | ------------------------------------------------------------------ |
| 🎧 **Modern Music Player Interface**             | Sleek appealing layout for seamless music experience.              |
| 📱 **Fully Responsive Design**             | Works perfectly across all devices and screen sizes.               |
| ⏯️ **Advanced Playback Controls**          | Complete control over your playlist with intuitive buttons.        |
| 🕒 **Dynamic Progress Tracking**          | Displays live playback time and track progress dynamically.        |

## <img src="https://github.com/KrishBharadwaj5678/Audioarc/raw/main/assets/readme/techStack.gif" width="35"/> Tech Stack

| Tech | Purpose |
|------|---------|
| <img src="https://skillicons.dev/icons?i=html" width="25"/> **HTML5** | Structure of the web pages and audio element integration. |
| <img src="https://skillicons.dev/icons?i=css" width="25"/> **CSS3** | Styling, animations, and responsive layout design. |
| <img src="https://skillicons.dev/icons?i=js" width="25"/> **JavaScript** | Handles music controls, progress updates, and interactivity. |
| <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/json/json-original.svg" width="25"/> **JSON** | Stores and manages song data like title, artist, and file paths. |
